# 快速使用指南 - V2.0 智能优化版

## 🎯 核心变化

**V2.0 不再需要手动设置每科目标分数！**

只需告诉系统：
1. 这门课最低能考多少分
2. 这门课最高能考多少分  
3. 这门课的难度如何

系统会自动用**运筹学算法**计算最优分数分配！

---

## 📝 5分钟上手

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

### 第二步：运行应用

```bash
python main.py
```

### 第三步：添加已修课程

在"已修课程"标签页：
- 输入课程名、学分、分数
- 点击"添加"

### 第四步：添加计划课程（重要！）

在"计划课程"标签页，填写：

**示例1：简单课程**
```
课程名：Web开发
学分：2.5
最低分：80  （即使不太努力也能拿到）
最高分：98  （全力以赴能达到）
难度系数：0.3  （简单课程）
```

**示例2：困难课程**
```
课程名：数据结构与算法
学分：3
最低分：70  （保守估计）
最高分：95  （乐观估计）
难度系数：0.8  （困难课程）
```

**难度系数参考：**
- 0-0.3：简单（通识课、选修课、擅长科目）
- 0.3-0.7：中等（一般专业课）
- 0.7-1.0：困难（核心专业课、数学、编程）

### 第五步：智能优化

1. 切换到"智能优化"标签页
2. 输入目标加权平均分（例如：87）
3. 点击"🚀 开始智能优化"
4. 查看结果！

## 📊 结果解读

### 优化成功 ✅

系统会显示：
- 每门课的**最优目标分数**
- 颜色编码：
  - 🟢 绿色：保持正常即可
  - 🟡 黄色：需要认真准备
  - 🔴 红色：需要全力以赴

**策略建议：**
- 简单课程 📗：重点提分，争取高分
- 中等课程 📘：正常准备
- 困难课程 📕：保证达标即可

### 优化失败 ❌

如果目标无法达成，系统会提供调整方案：

1. **降低目标GPA** - 例如从87降到85
2. **增加高分课程** - 多选几门简单的选修课
3. **提高最高分预期** - 重新评估自己的能力

## 💡 实用技巧

### 技巧1：如何估计最低分和最高分？

**最低分（保守估计）：**
- 问自己：即使发挥失常、准备不充分，能拿多少分？
- 参考：历年挂科率、课程难度

**最高分（乐观估计）：**
- 问自己：如果全力以赴、发挥出色，能拿多少分？
- 参考：往届学霸成绩、自己的强项

**建议范围：** 15-25分的跨度

### 技巧2：如何确定难度系数？

**方法1：主观评估**
- 这门课我擅长吗？→ 难度低
- 这门课挂科率高吗？→ 难度高
- 这门课需要大量时间吗？→ 难度高

**方法2：参考标准**
```
0.1-0.2: 非常简单（体育、艺术鉴赏）
0.3-0.4: 简单（通识选修、擅长科目）
0.5-0.6: 中等（一般专业课）
0.7-0.8: 困难（核心专业课、数学）
0.9-1.0: 非常困难（最难的课程）
```

### 技巧3：优化结果不满意？

**调整策略：**
1. 降低困难课程的难度系数（可能低估了自己）
2. 提高简单课程的最高分（可以更努力）
3. 调整目标GPA（更现实的目标）
4. 增加计划课程（多选课）

- 已修17.5学分，当前GPA 87.0

## ⚠️ 常见错误

### 错误1：最低分 ≥ 最高分
```
❌ 最低分：90，最高分：85
✅ 最低分：85，最高分：90
```

### 错误2：难度系数超出范围
```
❌ 难度：1.5
✅ 难度：0.8（0-1之间）
```

### 错误3：过于乐观的估计
```
❌ 数据结构：最低95，最高100
✅ 数据结构：最低70，最高95
```

### 错误4：所有课程难度都一样
```
❌ 所有课程难度都是0.5
✅ 根据实际情况区分难度
```

## 🔧 故障排除

### 问题：优化失败，提示"目标无法达成"

**原因：** 即使所有课程拿最高分也达不到目标

**解决：**
1. 降低目标GPA
2. 提高某些课程的最高分预期
3. 增加高分课程

### 问题：优化结果都是最高分

**原因：** 目标GPA设置过高

**解决：** 降低目标GPA到合理范围

### 问题：优化结果都是最低分

**原因：** 目标GPA设置过低

**解决：** 提高目标GPA，挑战自己
